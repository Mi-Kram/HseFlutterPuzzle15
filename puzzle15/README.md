# Puzzle15 - Flutter приложение

### Инструкция по запуску:
```
flutter pub get
flutter run
```

### Demo
[Демо видео](./docs/demo.mp4)


## Основной Функционал

### Игра

1. Генерация новых уровней в оффлайн режиме
1. Генерация уровней размером от 3x3 до 9x9.
1. Таймер и подсчёт количество ходов.
1. Плавное, анимированное передвижение плиток.
1. Кастомизация плиток (только числа, только картинка, числа и картинки).
1. Перемещение плиток с помощью мыши и стрелок на клавиатуре.
1. Когда экран головоломки не активен, таймер приостанавливается.
1. Выход из игры необходимо подтвердить.

### Ежедневные задания

1. Отрисовка интерактивного календаря с головоломками.
1. Получение ежедневных уровней по API.
1. Хранение лучших результатов каждого ежедневного уровня.

### Автосборка

1. Получение алгоритма для сборки любого уровня (по API).
1. Анимированное решение головоломки.
1. Регулирование скорости сборки головоломки.

### Прочее

1. Onboarding
1. Отдельная страница настроек:
   - Выбор темы: Системная, Светлая, Тёмная
   - Выбор типа плиток для головоломки:
     * только номер плитки
     * только картинка
     * и номер плитки, и картинка
   - Выбор картинки для плиток - можно загрузить свою картинку для отображения на головоломках


## Техническая реализация

1. Clean Architecture (Domain / Data / Presentation), feature-first
1. Dependency Injection: Flutter getIt
1. State Managment: Flutter bloc
1. Flutter tests
